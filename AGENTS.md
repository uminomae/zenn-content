# AGENTS.md (zenn-content)

このファイルを `zenn-content` での運用ルールの入口とする。

## このリポジトリの役割

- `zenn-content` は Zenn CLI 用の公開原稿リポジトリ
- 探索や長い試行錯誤の正本は別 repo / Issue にあってよい
- ここでは **公開用の最終原稿** を整える

## 記事を書く前に読むもの

1. `README.md`
2. `docs/WRITING-RULES.md`
3. 既存記事があれば `articles/` の近い原稿

## 基本方針

- 記事の主張は 1 本に絞る
- 背景、問い、読者にとっての意味を先に置く
- 一般ルールには案件固有の方針を書き込まない
- 時点依存の内容は、記事本文や草稿側で日付を明記する

## 他 repo から来たとき

- 別 repo で記事の構想を育てていてもよい
- `zenn-content` に来たら、公開原稿としての文体と構成は `docs/WRITING-RULES.md` に合わせる
- 記事固有の主題、代表例、注意書きは元 repo の Issue や草稿に残し、このファイルには持ち込まない
